
### 1. bcrypt
- Hashing passwords

### 2. cors
- cross origin resource sharing
- allows frontend running on diff domains or port to make request to your backend

### 3. dotenv
- loads environment variables from .env file into process.env managing sensitive information

### 4. express
- creating backend API, routing, middleware support, request, and response handling, and more

### 5. jsonwebtoken
- backend issues JWT upon a successful login, and client sends this token with subsequent requests to prove identity

### 6. mongoose
- object data modelling (ODM) lib for mongoDB
- higher level abstraction, define schemas, model based interaction with database

### 7. nodemon
- restarts node.js application when it detects file change
